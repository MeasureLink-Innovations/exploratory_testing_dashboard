# Screenshot Asset Conventions

This folder stores documentation screenshots used by `README.md` and `docs/architecture.md`.

## Naming convention

Use deterministic, flow-based names:

- `session-archive.png`
- `session-detail-timeline.png`
- `artifact-toolbelt.png`

For future additions:

- lowercase kebab-case
- include feature/flow intent in filename
- avoid timestamps in committed filenames

## Update rule

When a documented UI flow changes, update the corresponding screenshot in the same PR (or explicitly track a follow-up task).
